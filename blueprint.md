
# Blueprint: Lottery Number Generator

## Overview

This is a simple web application that generates and displays 6 unique lottery numbers between 1 and 45.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling.
*   `main.js`: The JavaScript file for functionality.

## Features

*   Generates 6 unique random numbers.
*   Displays the numbers in a visually appealing way.
*   A button to generate new numbers.

## Current Plan

*   **Task:** Create a lottery number generator website.
*   **Steps:**
    1.  Modify `index.html` to create the structure of the page.
    2.  Modify `style.css` to add styles.
    3.  Modify `main.js` to add the lottery number generation logic.
